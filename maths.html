1. Addition de fractions

Pour additionner deux fractions, les dénominateurs doivent être identiques.

Exemple : 1/4 + 2/4 = (1 + 2)/4 = 3/4

2. Soustraction de fractions

Comme pour l’addition, il faut que les fractions aient le même dénominateur.

Exemple : 3/5 - 1/5 = (3 - 1)/5 = 2/5

3. Multiplication de fractions

Multiplie les numérateurs entre eux, puis les dénominateurs entre eux.

Exemple : 2/3 × 4/5 = (2×4)/(3×5) = 8/15

4. Division de fractions

On inverse la deuxième fraction, puis on effectue une multiplication.

Exemple : 3/4 ÷ 2/5 = 3/4 × 5/2 = 15/8
