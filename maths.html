Chapitre : Les Fractions

Une fraction représente une partie d’un tout. Elle est constituée d’un numérateur (le haut) et d’un dénominateur (le bas).

Exemple : 3/4 signifie "trois parts sur quatre".

Types de fractions :

  • Fraction propre : numérateur < dénominateur (ex : 2/3)
  • Fraction impropre : numérateur > dénominateur (ex : 7/4)
  • Fraction apparente : numérateur divisible par le dénominateur (ex : 6/3 = 2)

Opérations sur les fractions

Addition & Soustraction :

Si les dénominateurs sont les mêmes, on additionne ou soustrait les numérateurs :

1/4 + 2/4 = 3/4

Si les dénominateurs sont différents, on les rend identiques :

1/2 + 1/3 = 3/6 + 2/6 = 5/6
Multiplication :
(2/3) × (4/5) = 8/15
Division :

On inverse la 2e fraction et on multiplie :

(2/3) ÷ (4/5) = (2/3) × (5/4) = 10/12
