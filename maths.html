Chapitre : Les Fractions

Addition de Fractions

Quand deux fractions ont le même dénominateur, on les additionne simplement en additionnant les numérateurs.

Exemple : 2/5 + 1/5 = (2+1)/5 = 3/5

Si les dénominateurs sont différents, il faut trouver un dénominateur commun.

Exemple : 1/4 + 1/6 = 3/12 + 2/12 = 5/12

Quiz :

Combien vaut 2/7 + 3/7 ?

Soustraction de Fractions

Les mêmes règles que pour l'addition s'appliquent, mais on soustrait les numérateurs.

Exemple : 5/8 - 2/8 = (5 - 2)/8 = 3/8

Quiz :

Combien vaut 6/9 - 2/9 ?

Multiplication et Division

Multiplication : On multiplie les numérateurs entre eux, et les dénominateurs entre eux.

Exemple : 2/3 × 3/4 = 6/12 = 1/2

Division : On multiplie la première fraction par l'inverse de la seconde.

Exemple : (3/4) ÷ (2/5) = (3/4) × (5/2) = 15/8

Quiz :

Combien vaut (1/2) × (3/5) ?
