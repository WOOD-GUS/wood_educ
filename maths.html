Chapitre : Les Fractions

Une fraction permet de représenter une partie d’un tout. Exemple : 1/2, 3/4, 5/6.

1. C’est quoi une fraction ?

Une fraction est composée de deux nombres :

  • Numérateur : le chiffre du haut
  • Dénominateur : le chiffre du bas
Exemple :
3/4
Cela signifie "3 parts sur 4".

2. Addition de fractions

Pour additionner, les dénominateurs doivent être les mêmes.

Exemple :
1/4 + 2/4 = 3/4

3. Soustraction de fractions

Comme pour l’addition, les dénominateurs doivent être identiques.

Exemple :
5/6 - 2/6 = 3/6

4. Multiplication de fractions

On multiplie les numérateurs entre eux, puis les dénominateurs.

Exemple :
2/3 × 3/5 = (2×3)/(3×5) = 6/15

5. Division de fractions

On garde la première fraction, on inverse la deuxième, puis on multiplie.

Exemple :
2/3 ÷ 4/5 = 2/3 × 5/4 = 10/12
