Chapitre : Les Fractions

Apprends à comprendre et à manipuler les fractions

Qu’est-ce qu’une fraction ?

Une fraction représente une part d’un tout. Elle est composée de deux éléments :

  • Le numérateur : le nombre de parts prises
  • Le dénominateur : le nombre total de parts
Exemple : 3/4 signifie que l'on prend 3 parts sur 4 égales.

Les différents types de fractions

  • Fraction propre : le numérateur est plus petit que le dénominateur (ex : 2/5)
  • Fraction impropre : le numérateur est plus grand que le dénominateur (ex : 7/4)
  • Fraction apparente : une fraction qui est égale à un entier (ex : 6/3 = 2)

Opérations sur les fractions

Addition et soustraction de fractions

Si les dénominateurs sont identiques, on additionne ou on soustrait les numérateurs.

Exemple : 1/4 + 2/4 = 3/4

Si les dénominateurs sont différents, il faut d’abord les rendre identiques (trouver un dénominateur commun).

Exemple : 1/2 + 1/3 = 3/6 + 2/6 = 5/6

Multiplication de fractions

On multiplie les numérateurs entre eux et les dénominateurs entre eux.

Exemple : (2/3) × (4/5) = 8/15

Division de fractions

On inverse la deuxième fraction et on effectue une multiplication.

Exemple : (2/3) ÷ (4/5) = (2/3) × (5/4) = 10/12

Simplification de fractions

On divise le numérateur et le dénominateur par un même facteur commun.

Exemple : 6/12 = 1/2
