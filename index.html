1. Addition de fractions

Pour additionner deux fractions, il faut qu'elles aient le **même dénominateur**. S'ils ne sont pas égaux, on les rend égaux (on les "met au même dénominateur") :

  • Exemple : 1/4 + 2/4 = (1+2)/4 = 3/4
  • Exemple : 1/2 + 1/3 → on transforme en 3/6 + 2/6 = 5/6

2. Soustraction de fractions

La soustraction suit le même principe : il faut que les dénominateurs soient identiques.

  • Exemple : 5/6 - 2/6 = (5-2)/6 = 3/6 = 1/2
  • Exemple : 3/4 - 1/2 → 3/4 - 2/4 = 1/4

3. Multiplication de fractions

Pour multiplier des fractions, on multiplie les numérateurs entre eux et les dénominateurs entre eux.

  • Exemple : 2/3 × 3/4 = (2×3)/(3×4) = 6/12 = 1/2
  • Pas besoin de mettre au même dénominateur !

4. Division de fractions

Diviser par une fraction revient à multiplier par son inverse :

  • Exemple : 2/3 ÷ 4/5 = 2/3 × 5/4 = 10/12 = 5/6
  • L'inverse de 4/5 est 5/4
